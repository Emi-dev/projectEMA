
Name of app is EMA music

Description - Application to search for artists or venues by location and also get info on artist.
Shows weather at the event for that date snd location
Pulls info on band, started, albums, members

As a music lover the person will want to go to site and search for a band to see where they are playing, or search for a venue to see what bands are playing.
A user will also see the bands info. A user will also see the weather for the date and area the band is playing.


API - Ticketmaster
API - weather
API - IMDB for music
